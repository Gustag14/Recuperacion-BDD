# Bases de datos: conceptos, usos y tipos

## Concepto
Una base de datos es una colección organizada de datos relacionados entre sí,
gestionada por un SGBD (Sistema Gestor de Bases de Datos).

## Tipos según el modelo de datos
- **Relacional**: organiza los datos en tablas (MySQL, PostgreSQL, SQLite)
- **Documental**: almacena documentos JSON/BSON (MongoDB)
- **Clave-valor**: pares clave → valor (Redis)
- **Grafos**: nodos y aristas (Neo4j)
- **Columnar**: columnas en lugar de filas (Cassandra)

## Tipos según la ubicación
- **Centralizada**: todo en un único servidor
- **Distribuida**: datos repartidos en varios nodos
- **En la nube**: hospedada en servicios cloud (AWS RDS, Azure SQL)
