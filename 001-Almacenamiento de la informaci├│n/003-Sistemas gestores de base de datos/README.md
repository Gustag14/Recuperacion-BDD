# Sistemas Gestores de Base de Datos (SGBD)

## Funciones principales
1. Almacenamiento persistente de datos
2. Control de acceso y seguridad
3. Gestión de transacciones (ACID)
4. Recuperación ante fallos
5. Control de concurrencia

## Componentes
- Motor de almacenamiento
- Procesador de consultas
- Gestor de transacciones
- Gestor de memoria
- Gestor de seguridad
