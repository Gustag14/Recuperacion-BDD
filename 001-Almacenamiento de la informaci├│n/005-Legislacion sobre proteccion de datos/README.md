# Legislación sobre protección de datos

## Marco legal (España / UE)
- **RGPD** (Reglamento General de Protección de Datos, UE 2016/679)
- **LOPDGDD** (Ley Orgánica 3/2018, España)

## Principios clave del RGPD
1. **Licitud**: tratar datos con base legal
2. **Limitación de finalidad**: solo para el fin declarado
3. **Minimización**: solo los datos necesarios
4. **Exactitud**: mantener los datos actualizados
5. **Limitación del plazo**: no guardar más tiempo del necesario
6. **Integridad y confidencialidad**: proteger contra accesos no autorizados

## Derechos del interesado
- Acceso, rectificación, supresión (derecho al olvido)
- Portabilidad, oposición, limitación del tratamiento
