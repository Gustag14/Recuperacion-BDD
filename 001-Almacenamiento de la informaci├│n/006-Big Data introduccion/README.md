# Big Data: introducción

## Las 5 V del Big Data
| V | Descripción |
|---|-------------|
| Volumen | Gran cantidad de datos (terabytes, petabytes) |
| Velocidad | Generación y procesamiento a alta velocidad |
| Variedad | Estructurados, semiestructurados y no estructurados |
| Veracidad | Calidad y fiabilidad de los datos |
| Valor | Beneficio que se extrae de los datos |

## Herramientas del ecosistema
- **Hadoop**: framework para procesamiento distribuido
- **Spark**: motor de análisis en memoria
- **Kafka**: mensajería en tiempo real
- **Hive**: consultas SQL sobre Hadoop

## Inteligencia de negocios (BI)
El BI transforma datos en información útil para la toma de decisiones.
Herramientas: Power BI, Tableau, Grafana.
