# Criterios de evaluación – Diagramas ER

## Lo que se evalúa

1. **Identificación de entidades** (2 puntos)
   - Detección correcta de todas las entidades del enunciado
   - Diferenciación entre entidades fuertes y débiles

2. **Atributos y claves** (2 puntos)
   - Asignación correcta de atributos a cada entidad
   - Identificación de la clave primaria de cada entidad

3. **Relaciones y cardinalidades** (3 puntos)
   - Nombre de cada relación
   - Cardinalidad correcta (1:1, 1:N, N:M)
   - Participación (total o parcial)

4. **Paso al modelo relacional** (2 puntos)
   - Creación correcta de tablas para entidades
   - Creación de tablas intermedias para relaciones N:M
   - Colocación correcta de claves foráneas

5. **Normalización** (1 punto)
   - El modelo resultante cumple al menos la 3FN

## Recomendaciones
- Leer el enunciado dos veces antes de empezar
- Identificar primero los sustantivos (entidades) y los verbos (relaciones)
- Comprobar cardinalidades con ejemplos concretos
